结论

  1. npm 失败后，可以加 –verbose 参数查看详细日志，定位问题。
  2. 编译源码可能你还需要装xcode-select --install
  3. 对于 Error: spawn .../node_modules/xxx/vendor/.. ENOENT,单独安装一下对应缺失module即可。
  4. node-gyp 和 libvips可能也会影响，建议重装一次
